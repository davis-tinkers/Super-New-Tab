Super New Tab

Why does this extension exist?
Edge (and Chrome) do not allow you to customize the new tab URL for "security reasons" which is really lame.
Super New Tab provides a simple toolbar button to open your favorite home page or local file in a new tab.
Simply click the toolbar button and a new tab will open with your desired home page or local file!

Note: Local file URLs (file:///) may not work due to browser security restrictions. If you use a local file:
  • Make sure the file path has no spaces (try renaming folders/files if needed).
  • Enable the extension's "Allow access to file URLs" permission in Edge's extension settings.
For best results, use a web page URL (e.g., https://zombo.com/).
